Аналіз алгоритмів

Number of 1 Bits 
Опис:

Підрахунок кількості одиничних бітів у числі без використання toString(2).

Алгоритм:
Беремо останній біт: n & 1
Зсуваємо число вправо: n >>> 1
Складність:
Час: O(k), де k — кількість бітів (до 32)
Пам'ять: O(1)

Reverse Bits
Опис:

Розвертає біти 32-бітного числа.

Алгоритм:
32 ітерації
Зсуваємо результат і додаємо останній біт
Складність:
Час: O(32) ≈ O(1)
Пам'ять: O(1)

Single Number
Випадок 1 (елементи по 2 рази)
Ідея:

Використання XOR:

a ^ a = 0
a ^ 0 = a
Складність:
Час: O(n)
Пам'ять: O(1)
Випадок 2 (елементи по 3 рази)
Ідея:

Використання двох змінних:

ones
twos
Складність:
Час: O(n)
Пам'ять: O(1)

Power of Two
Опис:

Перевіряє, чи є число степенем двійки.

Алгоритм:
n & (n - 1) === 0
⏱ Складність:
Час: O(1)
Пам'ять: O(1)

Bitwise AND of Numbers Range
Опис:

Обчислює побітове AND для всіх чисел у діапазоні.

Алгоритм:
Зсуваємо left і right вправо
Поки вони не стануть рівними
Складність:
Час: O(log n)
Пам'ять: O(1)
